# Ralph Progress Log
Started: Fri Jan  9 19:18:28 +08 2026

## Codebase Patterns
- This project uses Tailwind CSS v4 with `@tailwindcss/postcss` - no tailwind.config.js file needed
- Design tokens are defined in `src/app/globals.css` using CSS custom properties in `:root` and `@theme inline`
- Must use Node.js 20+ for builds: `export PATH="/opt/homebrew/Cellar/node@20/20.19.5/bin:$PATH"`
- Build command: `npm run build`
- Uses Next.js 16 with React 19
- Slate color palette maps to navy tones (slate-900 = #0f172a navy)
- Badge color pattern: bg-{color}-100 text-{color}-700 with rounded-full (teal for recommends, blue for needs)
- Card shadow pattern: shadow-md at rest, hover:shadow-lg with transition-shadow duration-200
- Header/navigation component is at `src/components/Navigation.tsx` (not Header.tsx)
- Premium search input pattern: h-[52px] sm:h-14, border-2 border-slate-200, rounded-xl, focus:border-teal-500 focus:ring-4 focus:ring-teal-500/20
- Category pills pattern: h-10 px-6 py-2 rounded-full, inactive: bg-white border-slate-200, active: bg-teal-600 border-teal-600
