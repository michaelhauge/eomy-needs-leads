## Codebase Patterns
- This project uses Tailwind CSS v4 with `@tailwindcss/postcss` - no tailwind.config.js file needed
- Design tokens are defined in `src/app/globals.css` using CSS custom properties in `:root` and `@theme inline`
- Must use Node.js 20+ for builds: `export PATH="/opt/homebrew/Cellar/node@20/20.19.5/bin:$PATH"`
- Build command: `npm run build`
- Uses Next.js 16 with React 19
- Slate color palette maps to navy tones (slate-900 = #0f172a navy)
- Badge color pattern: bg-{color}-100 text-{color}-700 with rounded-full (teal for recommends, blue for needs)
- Card shadow pattern: shadow-md at rest, hover:shadow-lg with transition-shadow duration-200
- Header/navigation component is at `src/components/Navigation.tsx` (not Header.tsx)
- Premium search input pattern: h-[52px] sm:h-14, border-2 border-slate-200, rounded-xl, focus:border-teal-500 focus:ring-4 focus:ring-teal-500/20
- Category pills pattern: h-10 px-6 py-2 rounded-full, inactive: bg-white border-slate-200, active: bg-teal-600 border-teal-600
- Filter dropdown pattern: h-10 rounded-full, inactive: bg-white border-slate-200, active: bg-teal-100 text-teal-700 border-teal-300, dropdown shadow-lg rounded-lg
- Form field pattern: h-[52px] border-[1.5px] border-slate-200 rounded-lg, focus:border-teal-500 focus:ring-2 focus:ring-teal-500/20, placeholder:text-slate-400
- Textarea pattern: min-h-[120px] with same styling as inputs
- Form label pattern: text-sm font-medium text-slate-700 mb-2
- Button system pattern: Primary: h-12 px-8 bg-teal-600 text-white text-base rounded-lg font-semibold, Secondary: bg-white border-slate-300 text-slate-700
- Button hover pattern: hover:bg-teal-700 hover:shadow-md (primary), hover:bg-slate-50 hover:shadow-md (secondary)
- Button active pattern: active:bg-teal-800 active:scale-[0.98] with transition-all duration-150
- Disabled button pattern: disabled:opacity-50 disabled:cursor-not-allowed
- Skeleton pattern: Use SkeletonCard component from `src/components/SkeletonCard.tsx` with animate-shimmer class
- Skeleton card sizing: min-h-[180px] sm:min-h-[200px], rounded-2xl to match actual cards
- SkeletonGrid component for rendering multiple skeleton cards (default count: 6)
